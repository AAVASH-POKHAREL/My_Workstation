What is Logical Operators in SQL?


SQL logical operators are used to test for the truth of the condition. A logical operator like the Comparison operator returns a boolean value of TRUE, FALSE, or UNKNOWN. In this article, we will discuss different types of Logical Operators. Logical operators are used to combine or manipulate the conditions given in a query to retrieve or manipulate data .there are some logical operators in SQL like OR, AND etc.

Some of the Operators are mentioned below with its syntax.

AND:
SELECT * FROM EMPLOYEE
WHERE SALARY < 10000 AND AGE>25

OR:
SELECT * FROM STUDENTS WHERE RANK >20 OR MARKS >600;

NOT :
SELECT * FROM EMP WHERE NOT SALARY =10000;


ORDER-Ing OPERATION

ASC:
SELECT * FROM EMP ORDER BY ID ASC;

DESC:
SELECT* FROM EMP ORDER BY ID DESC;

GROUP-ING operation


SELECT* FNAME ,lNAME FROM EMP GROUP BY Name;

AGGREGATE FUNCTIONS( MATHEMATICAL OPERATIONS)


AVERAGE:
SELECT avg(salary) FROM EMP;

MINIMUM:
SELECT MIN(salary) FROM EMP;

MAXIMUM:
SELECT max (salary) FROM EMP;

SUM:
SELECT sum(salary) FROM EMP;

COUNT:
SELECT (id) FORM EMP;

DISTINCT:
SELECT distinct(name) FROM EMP;

ROUND:
SELECT ROUND(MARKS,2) FROM STUDENTS;

String Functions ( String Handeing Functions)


LENGTH:
SELECT LEN(`APPLE`);

REVERSE:
SELECT REVERSE(`APPLE`);

REPLICATE:
SELECT REPLICATE(`APPLE`,S);

UPPERCASE:
SELECT UPPERCASE (`apple`);

LOWERCASE:
SELECT LOWERCASE (`APPLE);

CONCATENATION:
SELECT CONCATENATION (`NE’.’PA’,’L’);

LEFT MOST CHARACTER:
SELECT LEFT (`APPLE`,3);

RIGHT MOST CHARACTER:
SELECT RIGHT(`APPLE`,3);


OTHERS

IN:
SELECT ADDRESS FROM STD; WHERE IN (“KTM”,:BIKE”,”PTN”);

BETWEEN:
SELECT SALARY FROM EMP BETWEEN 3000 AND 40000;

PERCENTAGE:
SELECT NAME FROM STD NAME “%a”;

UNDERSCORE:
SELECT NAME FROM STD LIKE”%__B”

HAVING:
ELECT NAME FROM STD WHERE GRADE =”A” HAVING MARKS>800
